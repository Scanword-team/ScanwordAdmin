Автоматизировання система составления сканворда на заданную тему

Введение

Автоматизировання система составления сканворда на заданную тему будет использоваться для автоматизации процессов разгадывания, составления, редактирования сканвордов, составления и редактирования словарей понятий, составления галереи и медиатеки.

Программа использует базу данных PostgreSQL 14.5. Программная система совместима с Windows 7 и выше. Для пользования программой необходимо предустановить браузер - Google Chrome 86.0.4240.183 (64-битный) и выше или Firefox 83.0 (64-битный) и выше.

Авторизация, запуск программы

Данная программная система поддерживает две роли пользователей: роль администратора и роль игрока. При запуске программной системы открывается страница авторизации, представленная на рисунке. Необходимо ввести логин и пароль в соответствующие поля, после чего пользователь может нажать кнопку "Зарегистрироваться" или "Войти" для регистрации новой учётной записи игрока или входа в систему соответственно.

Если данные при регистрации введены верно, то система уведомляет пользователя об успешной регистрации новой учётной записи. В случае неудачи система выдаёт сообщение о введённых некорректных данных.

Если данные при входе введены корректно и пользователь с такой учетной записью зарегистрирован, то система определяет роль пользователя и отображает соответствующий интерфейс. На рисунках представлены интерфейсы пользователя после успешной авторизации в роли администратора и игрока.

Если данные при входе введены некорректно, то система уведомляет пользователя об ошибке.

Администратор

Основная задача администратора – составление сканворда. Перед составлением сканворда он должен настроить его параметры: задать размеры сетки по вертикали (от 10 до 20 ячеек) и по горизонтали (от 10 до 20 ячеек), задать количество подсказок (от 1 до 3), подключить словарь понятий.

По заданным параметрам система должна создать шаблон сканворда (сетку), после этого можно будет перейти к заполнению сканворда заданиями.

При ручном создании администратор должен выбрать ориентацию слова (по горизонтали, по вертикали), затем выбрать слово из списка доступных, переместить его на сетку. Система в начале слова будет располагать “?”, в который будет помещено само задание.

При удалении слова из сетки администратор должен взять слово и переместить его обратно в словарь. После создания сканворда администратор сможет сохранить сканворд в базу данных.

При работе со словарями понятий администратор имеет возможность: добавления, изменения и удаления понятий, возможность выбора критерия сортировки (2 критерия сортировки – по алфавиту и по длине слова), возможность ввода маски для поиска, загрузки словаря из файла, сохранения словаря из файла и создания нового словаря понятий.

При работе с галереей администратор может добавлять картинки, просматривать и удалять их.

При работе с медиатекой администратор может добавлять мелодии, просматривать и удалять их.

Создание сканворда

Для создания нового сканворда администратору необходимо выбрать пункт меню "Создание сканворда". После этого на странице отобразится выпадающий список, в котором администратор может выбрать подходящий ему словарь понятий, числовые поля для задания количества ячеек по вертикали и горизонтали (минимальное - 10, максимальное - 20), и числовое поле для задания количества подсказок (минимальное - 1, максимальное - 3). После задания параметров сканворда администратор нажимает на кнопку "Перейти к составлению сканворда".

Система отображает страницу "Составление сканворда", представленную на рисунке, с заданными параметрами сканворда.

В правой части этой страницы расположен список понятий, который подгружает система в соответствии с выбором администратора, в центральной части находится сама сетка сканворда, размеры которой система определяет по заданным администратором значениям. В процессе составления сканворда администратор имеет возможность изменить размеры сетки, чтобы не возвращаться к заданию исходных параметров, если ему это необходимо. Для этого внизу слева есть числовые поля, отведённые для изменения сетки сканворда. При изменении сетки, все слова, находившиеся на сетке, автоматически удаляются из неё и возвращаются обратно в словарь.

Для добавления слова на сетку достаточно взять слово и перетащить её на сетку в ячейку, с которого начнётся это слово. В эту ячейку добавится символ "?", а в следующие ячейки по вертикали или по горизонтали, в зависимости от активной радиокнопки радиогруппы, которая располагается над словарём, будет добавлено слово. Система при взятии слова из словаря подсвечивает зелёным цветом доступные места для расположения и красным - ячейки, в которые невозможно добавить слово. Для удаления слова из сканворда достаточно взять за любую ячейку, относящуюся к слову, и перетащить её в область словаря. В случае успеха система очистит ячейки и добавит это слово обратно в словарь. Также имеется возможность фильтрации словаря: по наличию картинки, по наличию мелодии.

После того, как сканворд готов, администратор нажимает кнопку "Сохранить сканворд". И система отправляет новый объект Сканворд на сервер для дальнейшего его добавления в базу данных.

Редактирование сканворда

Для редактирования сканворда администратору необходимо выбрать пункт меню "Редактирование сканворда". В случае если список сканвордов пуст, то администратору выдастся соответствующее сообщение. Иначе система отобразит список сканвордов, имеющихся в базе данных.

При нажатии на элемент списка система получает соответствующий объект Сканворд и отображает страницу "Редактирование параметров сканворда". Наполнение страницы совпадает с наполнением страницы "Создание сканворда". Значения параметров (количество ячеек по вертикали и по горизонтали, количество подсказок) устанавливаются такие, какие относятся к данному объекту. Причём изменение размеров сетки на данном этапе заблокировано, чтобы администратор не потерял расположение понятий на сетке. После редактирования параметров сканворда администратор нажимает на кнопку "Перейти к составлению сканворда".

Система отображает страницу "Редактирование выбранного сканворда". Её наполнение совпадает с наполнением страницы "Составление сканворда". На сетке при отображении страницы система располагает понятия, которые были добавлены при создании сканворда.

После того, как редактирование сканворда закончено, администратор нажимает кнопку "Сохранить сканворд". И система отправляет изменённый объект Сканворд на сервер для дальнейшего его обновления в базе данных.

Работа со словарём понятий

Для работы со словарём понятий администратору необходимо выбрать пункт меню "Словарь понятий". Система отображает страницу "Словарь понятий". Сверху располагается блок для создания нового словаря. В нём находится текстовое поле для ввода названия словаря, кнопка «Выберите файл» для подключения внешнего словаря понятий. А также кнопка «Создать» для создания нового словаря в базе данных. Изначально она недоступна, пока не введено название словаря Под ним находится блок для выбора имеющегося словаря в базе данных. В этом блоке располагается выпадающий список с названиями словарей и кнопка «Подтвердить», которая также недоступна, пока не выбран словарь.

При нажатии на кнопку "Создать" или "Подтвердить" система отображает таблицу с наполнением словаря. Таблица может содержать элементы, состоящие из двух полей - определение и описание понятия, и кнопки, при нажатии на которую соответствующий элемент будет удаляться из списка. Над таблицей располагаются две кнопки и одно текстовое поле: кнопка «+» для создания нового понятия, при нажатии на которую открывается модальное окно; кнопка для сохранения словаря в базу данных; текстовое поле для поиска вопроса по маске. Для редактирования элемента таблицы необходимо нажать на соответствующий элемент, после этого откроется модальное окно.

Модальное окно имеет два текстовых поля (определение и описание понятия), две радиокнопки: картинка и аудио для выбора типа задания. Под ними находится кнопка "Открыть галерею" или "Открыть медиатеку" в зависимости от состояния радиогруппы, при нажатии на которую открывается галерея или медиатека соответственно, для выбора медиафайла. Под кнопкой располагается уже имеющийся медиафайл с кнопкой "X" для его удаления. Внизу модального окна располагаются кнопки "Сохранить" и "Отмена" для сохранения или отмены изменения соответственно.

В случае, если такое определение уже есть в словаре или не все текстовые поля были заполнены, при нажатии на кнопку "Сохранить" система выдаёт сообщение об ошибке.

Работа с галереей

Для работы с галереей администратору необходимо выбрать пункт меню "Галерея". Система отображает страницу "Галерея". На ней находится список элементов, содержащих изображение и кнопку «Удалить», при нажатии на которую удаляется соответствующий элемент. Над списком находится кнопка «Добавить» – для добавления нового изображения в список. В случае если содержимое файла некорректно, система выдаёт соответствующее сообщение.

Работа с медиатекой

Для работы с медиатекой администратору необходимо выбрать пункт меню "Медиатека". Система отображает страницу "Медиатека". На ней находится список элементов, содержащих мелодию и кнопку «Удалить», при нажатии на которую удаляется соответствующий элемент. Над списком находится кнопка «Добавить» – для добавления новой мелодии в список. В случае если содержимое файла некорректно, система выдаёт соответствующее сообщение.
